# System Requirments
- Node.js v 8.9.2 ++
- NPM or Yarn

# Installation
clone this project.
```sh
$ git clone https://github.com/imgrbs/watch-together
```
install modules
```sh
$ npm install
```
or using yarn
```sh
$ yarn install
```
copy .env file
```sh
$ cp .env.example .env
```
note - you can edit port for deploy in .env file

build project
```sh
$ npm run build
```
or using yarn
```sh
$ yarn build
```

the last, run server and enjoy!
```sh
$ node server.js
```
